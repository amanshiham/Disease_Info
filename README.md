# Disease Info Website Model

A comprehensive, responsive web platform designed to provide reliable information on various diseases, empowering users through knowledge and accessible healthcare tools.

## 🚀 Features

*   **Disease Education Hub**: Detailed sections on conditions like **Sickle Cell Anemia**, **Hypertension**, and **Asthma**, covering symptoms, causes, and treatments[cite: 4, 5].
*   **Interactive Healthcare Tools**:
    *   **BMI Calculator**: Real-time Body Mass Index calculation with health-status alerts[cite: 5].
    *   **Appointment Booking**: Integrated booking system that generates unique tracking tokens for users[cite: 2].
*   **Modern Visuals**: Cinematic aesthetic with a responsive Bootstrap-based UI and smooth CSS animations[cite: 1, 4].
*   **Multimedia Integration**: Embedded video backgrounds and 3D model viewing capabilities for enhanced learning[cite: 5].
*   **Contact & Community**: Direct links for consultation and social media integration via WhatsApp, Instagram, and Facebook[cite: 3].

## 🛠️ Tech Stack

*   **Frontend**: HTML5, CSS3 (Custom Animations & Gradients)[cite: 1, 2, 5].
*   **Framework**: Bootstrap 4.5[cite: 1, 4].
*   **Scripting**: JavaScript (ES6+), jQuery[cite: 2, 5].
*   **Icons**: Font Awesome[cite: 3].

## 📂 Project Structure

The project is organized into modular files to ensure clean navigation and consistent styling:

*   **`main.html`**: The central landing page featuring disease categories[cite: 4].
*   **`about.html`**: Contains the project mission and introduces the development team.
*   **`booking.html`**: Dedicated interface for scheduling health consultations with token generation[cite: 2].
*   **`sicklecell.html`**: Detailed educational page utilizing a video background and BMI calculator[cite: 5].
*   **`contact.html`**: Directory of communication channels including email and social media[cite: 3].
*   **`styles.css`**: Primary stylesheet defining the global visual theme[cite: 6].
